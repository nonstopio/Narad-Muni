Install on macOS

The sage's scroll is unsigned. Follow these steps after downloading:

1
Open the .dmg file and drag Narad Muni into Applications.
2
Remove the quarantine flag by running this in Terminal:
xattr -cr "/Applications/Narad Muni.app"
3
Open the app normally. Narayan Narayan!

Install on Windows

Windows SmartScreen may block the unsigned installer:

1
Run the downloaded .exe installer.
2
If SmartScreen appears, click "More info".
3
Click "Run anyway" to proceed. Narayan Narayan!
